Snackbar 消息条

用于显示弹出消息。引入这个元素会同时引入一个全局函数 aqSnackbar。

你可以在控制台试试下面的代码。

aqSnackbar('Message') aqSnackbar({message: 'Message', type: 'success', placement: 'top-right'})

这个函数的详细信息：

interface AqSnackbarProperties { message: string | HTMLElement, type: 'info' | 'success' | 'warning' | 'error', placement: 'top-left' | 'top-center' | 'top-right' | 'bottom-left' | 'bottom-center' | 'bottom-right', html: boolean, duration: number, icon: string, closeable: boolean } declare function aqSnackbar(message: string) declare function aqSnackbar(prop: AqSnackbarProperties)

显然AqSnackbarProperties里面的属性也是这个组件上的属性。

Slot

名称 描述
[默认] 消息内容
icon 图标
action 除了关闭消息之外的动作，例如打开某个其他东西
close 自定义关闭按钮
