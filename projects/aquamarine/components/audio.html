Audio 音频

本质上就是把原生的 audio 元素套了一层壳。但是也有一些新功能。

属性

原生 audio 元素上的大多数属性在这上面都有，这里介绍最重要的属性。

名称 描述 类型
src 音频文件的地址 string
thumbnail 封面图片的地址 string
loop 是否循环播放 boolean
downloadable 是否提供下载按钮 boolean

该说不说
