CSS 锚点定位

现在你可以将一个元素直接相对于某个其他元素进行定位，所以不再需要 JS 来制作悬浮提示或者输入建议了。

演示

Hello!

代码

<div style="position: relative;"> <button class="anchor-1">A Button</button> <div class="tooltip">Hello!</div> </div> <div style="position: relative;" class="input-wrapper"> <label for="demo">Input</label> <input type="text" name="demo" id="demo"> <div class="suggestion"> <button>Option1</button> <button>Option2</button> </div> </div> .anchor-1 { anchor-name: --anchor-1; } .tooltip, .suggestion { position: absolute; top: anchor(bottom); left: anchor(start); width: anchor-size(width); background-color: var(--background-color); box-shadow: var(--general-shadow); display: none; z-index: 10; transition: 0.2s allow-discrete; opacity: 0; @starting-style { opacity: 1; } } .tooltip { position-anchor: --anchor-1; margin-top: 0.2rem; } #demo { anchor-name: --input; } .suggestion { position-anchor: --input; } .suggestion button { display: block; width: 100%; } .anchor-1:is(:hover, :focus) + .tooltip, .input-wrapper:focus-within .suggestion, .suggestion:is(:hover, :focus) { display: block; opacity: 1; @starting-style { opacity: 0; } }
