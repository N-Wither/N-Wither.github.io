属性列表
属性 描述 类型
format 时间格式，其中的 hh mm ss ms 会被分别替换为小时、分、秒、毫秒，其余字符不变。默认为 mm:ss。 string
time 起始时间，以毫秒计。默认为 0。 number
countdown 控制是否为倒计时。 boolean
step 控制文本的刷新间隔，以毫秒计，默认为 10。可用于减少性能消耗。 number
